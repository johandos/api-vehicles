## Api Vehiculos

Ejercicio de realizado en symfony en el cual se hace un CRUD basico de vehiculos

1. docker-compose build
2. docker-compose up -d
3. docker exec -it app-symf
4. cd vehicle 
5. composer install
6. ./vehicle/bin/console doctrine:migrations:migrate
